# Векторный графический редактор

Простая консольная программа для создания и управления геометрическими фигурами: точками, отрезками, кругами и квадратами.

## Возможности

- Создание геометрических фигур:
  - Точки (координаты x, y)
  - Отрезки (координаты начала и конца)
  - Круги (координаты центра и радиус)
  - Квадраты (координаты левого верхнего угла и длина стороны)
- Просмотр списка всех созданных фигур
- Перемещение фигур (изменение их координат)
- Удаление фигур по ID
- Интерактивный консольный интерфейс

## Структура классов

### Point (Точка)
Базовый класс для представления точки на плоскости.
- Атрибуты: `x`, `y` (координаты)
- Методы: перемещение, получение и установка координат

### Segment (Отрезок)
Представляет отрезок между двумя точками.
- Атрибуты: `start`, `end` (точки начала и конца)
- Методы: перемещение, вычисление длины

### Circle (Круг)
Представляет круг с центром и радиусом.
- Атрибуты: `centre` (центр), `radius` (радиус)
- Методы: перемещение, изменение радиуса

### Square (Квадрат)
Представляет квадрат со стороной, параллельной осям координат.
- Атрибуты: вершины (левая верхняя, правая верхняя, левая нижняя, правая нижняя), длина стороны
- Методы: перемещение

### VectorEditor (Редактор)
Основной класс управления приложением.
- Хранение списка фигур
- Добавление и удаление фигур
- Перемещение фигур
- Интерактивный интерфейс командной строки

## Использование

Запустите программу:
```bash
python vector_editor.py
```

### Доступные команды

```
add point <x> <y>                    - создать точку
add segment <x1> <y1> <x2> <y2>       - создать отрезок
add circle <x> <y> <radius>           - создать круг
add square <x> <y> <side_length>      - создать квадрат
list                                  - показать все фигуры
move <id> <параметры>                  - переместить фигуру
remove <id>                            - удалить фигуру
help                                   - показать справку
exit                                   - выход из программы
```

### Примеры использования

```
> add point 10 20
Создана точка с ID 1

> add segment 0 0 100 100
Создан отрезок с ID 2

> add circle 50 50 25
Создан круг с ID 3

> add square 10 10 30
Создан квадрат с ID 4

> list
Список фигур
ID: 1, Тип: Точка, Фигура: Точка (10.0, 20.0)
ID: 2, Тип: Отрезок, Фигура: Отрезок [(0.0, 0.0) -> (100.0, 100.0)]
ID: 3, Тип: Круг, Фигура: Круг (центр: (50.0, 50.0), радиус: 25.0)
ID: 4, Тип: Квадрат, Фигура: Квадрат (Левая верхняя: (10.0, 10.0), Правая верхняя: (40.0, 10.0), Левая нижняя: (10.0, 40.0), Правая нижняя: (40.0, 40.0)), длина стороны: 30.0

> move 1 15 25
Точка 1 перемещена

> remove 2
Удалена фигура с ID 2

> exit
До свидания!
```
